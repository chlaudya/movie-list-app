# Movie List App

### How to run

1. Clone the repo
   ```sh
   git clone https://github.com/chlaudya/movie-list-app.git
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Runs the app in the development mode
   ```sh
   yarn start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
